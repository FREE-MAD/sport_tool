# 小程序项目

一个纯净的微信小程序项目。

## 项目结构

```
├── miniprogram/
│   ├── pages/
│   │   └── index/       # 首页
│   ├── app.js           # 小程序入口
│   ├── app.json         # 全局配置
│   ├── app.wxss         # 全局样式
│   └── sitemap.json     # 站点地图
└── project.config.json  # 项目配置
```

## 开始开发

1. 使用微信开发者工具打开本项目
2. 在 `pages/` 目录下创建新页面
3. 在 `app.json` 的 `pages` 数组中注册新页面
